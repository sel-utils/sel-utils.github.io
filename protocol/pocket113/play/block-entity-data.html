pocket113 / play

Block Entity Data

Id: 56

Hex: 38

Bin: 00111000

Sent by the server and the client

Sets a block entity's nbt tag, block's additional data that cannot be indicated in the block's meta. More informations about block entities and their tag format can be found on Minecraft Wiki.

The client sends this packet when it writes a sign.

Fields:

Name Type
position blockPosition
nbt bytes
  • position

    Position of the block that will be associated with tag.

  • nbt

    Named binary tag of the block. The format varies from the classic format of Minecraft: Pocket Edition (which is like Minecraft's but little endian) introducing the use of varints for some types:

    • The tag `Int` is encoded as a signed varint instead of a simple signed 32-bits integer
    • The length of the `ByteArray` and the `IntArray` is encoded as an unsigned varint instead of a 32-bits integer
    • The length of the `String` tag and the named tag's name length are encoded as an unisgned varint instead of a 16-bits integer
