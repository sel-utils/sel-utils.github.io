hncom2 / status

Remote Command

Id: 14

Hex: 0E

Bin: 00001110

Sent by the server

Executes a command on the node.

Fields:

Name Type When
origin ubyte
sender address origin is not equal to 0
command string
command id varint
  • origin

    Origin of the command. It could be the hub itself or an external source.

    Constants:

    NameValue
    hub 0
    external console 1
    rcon 2
  • sender

    Address of the sender if the command has been sent from an external source and not the hub.

  • command

    Commands and arguments that should be executed on the node. For example say hello world or transfer steve lobby12.

  • command id

    Identifier of the command. It's sent in Log.commandId if the command generates an output.
