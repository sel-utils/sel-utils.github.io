hncom2 / status

Reload

Id: 16

Hex: 10

Bin: 00010000

Sent by the server

Notifies the node that the hub's reloadeable settings have been reloaded and that the node should also reload its resources (for example plugin's settings).

The fields of the packet may be empty if not updated during the reload.

Fields:

Name Type
display name string
motds motd[]
language string
accepted languages string[]
social json string
  • display name

    Display name of the server, same as HubInfo.displayName.

  • motds

    New MOTDs (message of the day) for the supported games.

  • language

    Main language of the server, in the same format as HubInfo.language.
